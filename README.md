## 스터디 프로젝트 구조

```
  .
  ├── LECTURE_JAVA com.study.algo.src
  │    │    : 자바(Java) 알고리즘 문제풀이
  |    ├── do_it
  |    │    : Do It 자료구조 알고리즘 자바편 실습
  |    ├── etc
  |    │    : 그외 유투브 인강 또는 인터넷 자료
  |    └── inflearn/basic
  |         : (인프런) 자바(Java) 알고리즘 문제풀이 입문: 코딩테스트 대비
  ├── LECTURE_PYTHON
  │    : Python 자료구조/알고리즘 (패스트캠퍼스)
  ├── ONLINE_CODING_TEST
      ├── codility
      │    : 코딜리티 문제
      ├── nojam
      │    : 백준 온라인 저지 문제
      └── school.programmers
           : 프로그래머스 문제
```
